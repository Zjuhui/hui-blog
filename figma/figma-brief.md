# Figma Brief

## Core Signal

- Name: 珲
- Mood: hard sci-fi, restrained academic, personal but not cute
- First viewport: a large calligraphic “珲” and the line “If You Are Nothing Without AI, Then You Shouldn't Have It!”
- Structure: Research / Life / Works / Archive
- Figma file: https://www.figma.com/design/KcZFkNsH1kX69UJdzT9mwz
- Clean pass spec: `clean-pass-spec.md`

## Visual Direction

- Background: near-black
- Text: warm paper white
- Accents: electric cyan, oxidized red, signal green
- Typography:
  - English: heavy grotesk for headline, neutral sans for interface
  - Chinese: neutral sans for body, calligraphic serif/script only for the logo glyph

## Components To Design First

1. Wordmark and avatar based on “珲”.
2. Top navigation.
3. Home hero.
4. Post card.
5. Article template.
6. Token ledger entry.

## Avatar Ideas

- Round avatar with a cropped brush-style “珲”.
- Square seal version using red accent.
- Monochrome favicon version using only the character silhouette.

## Page Notes

- Home should feel like an entry hatch, not a marketing page.
- Research pages can be denser and more technical.
- Life pages can have softer spacing and longer prose rhythm.
- Works should preserve project context: motivation, artifact, version, reflection.
